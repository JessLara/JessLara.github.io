Problem

Online cloud storage demands are increasing on a daily basis with more people accessing the internet and requiring to store their data in a place that’s safe and accessible from anywhere.

Solution

ByteSpace will create an organized environment, also known as a space, where you can keep files and content in one quick and easy to access place. ByteSpace is a tool that will allow users to create content quickly; whether that’s a space to keep notes, or a space to share photos, everything is just a click away.

Research

Cloud and non-cloud storage user are in need of a service that will help them stay organize; they want a place where they can create content, organize their files and where they can collaborate with other users. Current cloud storage users are looking for a service that provides them with a large amount of storage space, file creation and collaborative tools.

User Survey Summary

76% Surveyed use a cloud storage service
45% Surveyed found accessing their files was problematic with their current storage services
75% Surveyed wanted larger amount of free storage

Cloud Storage Users

SWOT Analysis

A SWOT analysis was completed with three competitors.

Google Drive | DropBox | Bear

The analysis indicated that usability, cross-platform accessibility and storage were critical. Along with that the importance of an intuitive, easy to use design was strongly required. Within the cloud storage market there is a need for an application like ByteSpace, users want more storage space. Creating an application that can compete with other services by providing a competitive (larger amount of free space) will be key in the foundation of ByteSpace.

User Stories

User stories consisted of three user roles:

  • A new user – New Bytespace user
  • Returning user – Bytespace user
  • Administrator – ByteSpace team account

User stories were focused on the feedback that was received on my initial survey along with the follow up survey for cloud storage users. To view the full list of user stories here for a full view of all user stories.

User Flows

The userflows were worked through high priority users process, I focused on the main need when first accessing ByteSpace, login in and uploading a file.

Wireframes

When creating the sketches keeping the users need in mind was important. Users want a place where they can easily collaborate, create content and share files/content. With that in mind, in the homepage there was a highlight on the collaboration aspects as well as testimonials and a login/sign up option. With the dashboard I wanted the user to easily be able to access, upload and view their files.

Wireframes Sketches

Low Fidelity

After sketching multiple different pages following the process of high fidelity flowcharts, I used Sketch to create the low fidelity illustration my design.

High Fidelity

High Fidelity designs were created with the user in mind, preference testing, along with user testing were conducted on multiple phases of the design process.

Iterations

Throughout the process there were multiple updates and changes to the ByteSpace design, with the aide of user testing, knowing what worked and didn’t helped developed ByteSpace and will continue to do so.

Branding

ByteSpace as a brand illustrates technology, clarity and functionality. A motherboard was used as an inspiration. The logo was inspired by the circuitry on a motherboard.

Having the opportunity to create a service from the beginning to the end was a tremendous learning experience, learning how the research leads to creating an entire brand, its design and its future development was a humbling and fulfilling experience.
